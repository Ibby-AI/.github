# Ibby AI

We're on a mission to bring agents to everyone.

That means building **agents** people can use, and the **infrastructure** developers need to run them — reliably, in the background, at scale.

<hr />

## What we build

**Agents** — AI capable of completing economically valuable tasks, with minimal supervision from humans.

**Agent infrastructure** — opinionated building blocks and plumbing for running agents in production at scale.

<hr />

## First release: Rafiki

[**Rafiki**](https://github.com/Saidiibrahim/rafiki) is an open-source agent harness for running remote background agents. Built on [Modal](https://modal.com) and Cloudflare, it gives developers a customisable foundation for deploying stateful, long-running agents.
